Key Features

✔️ User-Friendly Booking Forms - Enter your destination, travel dates, and number of guests effortlessly.
✔️ Interactive Date Picker – Select travel dates with ease using a modern calendar interface.
✔️ Responsive UI – A sleek, professional design that works seamlessly on desktops, tablets, and mobile devices.
✔️ Elegant Aesthetics – Monochrome travel-themed visuals with a professional touch.
✔️ Secure & Fast – Ensures a quick booking process without unnecessary steps.

Simon Says Game

Introduction

Simon Says is a classic memory-based game that tests your ability to remember and follow patterns. The game increases in difficulty as the sequences get longer, making it an engaging way to challenge your memory skills.

Key Features

✔️ Colorful Game Interface – Bright, vibrant colors make the game visually appealing.
✔️ Interactive Sound Effects – Each button press produces unique sound feedback.
✔️ Dynamic Difficulty – The sequence gets longer with each correct attempt.
✔️ Game Over Notification – Notifies the user of their final score and allows restarting.
✔️ Responsive & Smooth Animations – Works seamlessly on both desktop and mobile.

Why I Built This

I wanted to create a fun and interactive game that challenges players in an engaging way. Simon Says is a great way to sharpen memory and reaction time, and I enjoyed bringing this nostalgic game to life.

Challenges & Learnings

Managing Game Logic: Ensuring correct sequence recognition and progression.
UI & Sound Synchronization: Making sure colors and sounds align perfectly.
Performance Optimization: Keeping the game smooth across all devices.
